# Blog Django
Sistema web de postagem de notícias que realiza operações integrais de CRUD com banco de dados.

## Ferramentas utilizadas
- Python 3.5.2
- Django 1.8.5 com módulos Whitenoise e MySQLClient
- MySQL 5.7.18
- HTML5 e CSS3
- Bootstrap 3
